## 如何使用
  1. gitclone本仓库到你的hexo主题文件夹下
   
   ```git
   git clone https://github.com/xingwangzhe/hexo-theme-wang.git
   ```
   或者使用npm安装到hexo的依赖中
   
   ```bash
   npm install hexo-theme-wang
   ```
  2. 在hexo更目录下的_config.yml中更改使用主题
   ```yml
   theme: hexo-theme-wang
   ```
  3. 运行hexo s 查看效果
